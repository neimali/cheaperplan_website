# CheaperPlan Website

CheaperPlan 的官方静态网站，包含用户邮箱验证欢迎页面、隐私政策和使用条款。

## 页面

- `/` - 首页
- `/welcome` - 邮箱验证成功欢迎页面
- `/privacy` - 隐私政策
- `/terms` - 使用条款

## 本地开发

```bash
npm install
npm run dev
```

## 部署到 Vercel

1. 将项目推送到 GitHub
2. 在 Vercel 中导入项目
3. Vercel 会自动检测 Next.js 项目并进行部署

## 技术栈

- Next.js 14
- TypeScript
- 静态导出（Static Export）
- 响应式设计
